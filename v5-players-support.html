Version 5 players will no longer be supported within Video Cloud Studio and will not be displayed within the UI. These players will still playback content and are embeddable. However, editing or updating these players using Video Cloud Studio will not be possible.

The one exception will be the Brightcove Default Player (the player that every account comes with). You can delete this player through the Players module in Video Cloud Studio, and our Brightcove System will automatically generate a new Brightcove Default Player with the latest version.

If you want to know if your account still contains version 5 players, you can use the Display Players sample app.

It is still possible to update the player version using the Player Management API or, with this sample app, Update Player Version. However, you will not be able to publish a player that still got a 5 version using this API. The version 5 players in production (and their embed codes) will continue working as expected.

We highly recommend deleting the players with version 5 and creating new players to avoid confusion.
